TMNSearch

Description:
  • The TRACKMENOT.TMNSearch object, used as a high-level interface and manager for the core search functionality of the extension. Created in trackmenot.js.
Source:
  • trackmenot.js, line 59
Properties:
Name Type Description
tmn_tab_id number the ID of the tab controlled by TMN for searches
useIncrementals Boolean constant setting for random possibility to search about the same topic / dummy query multiple times ("incremenetal" searches)
incQueries array variable used to set an incremental sequence of searches about the same topic / dummy query
engine string the current search engine to search on
prev_engine string the last engine used to send a search
tmn_engines object an object containing each engine known to TMN, whether it is enabled for searches, and associated metadata (e.g. the engine's base search URL)
TMNQueries object the dummy query strings for TMN to search with, organized by keys representing the method of query generation typeoffeeds ("rss", "extracted", "dhs")
zeit_queries array a preset array of query terms representative of a "zeitgeist"
typeoffeeds array array of keynames for the dummy query generation methods in use
tmnLogs array running log of all dummy queries searched, used to update local storage for display of search logs in options page
burstEngine string the consistent search engine name for a sequence of "bursted" searches
burstTimeout number value in milliseconds to wait between searches in a "burst"
burstCount number the number of searches sent within a sequence of "bursted" searches
tmn_options object contains the user-configured options of TMN extension, restored from local storage between sessions
tmn_searchTimer the setTimeout timer for scheduling the next search
tmn_mode string the current search function mode - from the set {"timed", "recovery" (retrying when a search fails), or "burst"}
tmn_errTimeout (deprecated - unused) a setTimeout timer 3*the search timeout to initiative error
tmn_scheduledSearch Boolean status indicator for whether the next search has been scheduled
tmn_hasloaded Boolean status indicator for whether or not the search results page from the current search has loaded
currentTMNURL string the current search URL
currentUrlMap deprecated/unused
debug_ deprecated/unused
tmn_logged_id id of the last logged search
last_log_id id of the last log request's search
