Description:
  • The TRACKMENOT.TMNInjected object, which manages the search behavior and user simulation within TMN's tab, or in the background in "stealth" search mode. Created in tmn_search.js
Source:
  • tmn_search.js, line 36
Properties:
Name Type Description

tmnCurrentURL string the current search URL, used to keep track of the current search and for message handling/passing with the high-level TMNSearch object in trackmenot.js
engine string the search engine used in the current search, parsed from the parameter set by TMNSearch
last_engine string deprecated/unused
