Module: TMNInjected

A function to create the TRACKMENOT.TMNInjected object, which manages the search behavior and user simulation within TMN's tab, or in the background in "stealth" search mode
Properties:
Name Type Description
current_request_id number the id of the current search request, to prevent duplicate searches
tmnCurrentURL string the current search URL, used to keep track of the current search and for message handling/passing with the high-level TMNSearch object in trackmenot.js
engine string the search engine used in the current search, parsed from the parameter set by TMNSearch
